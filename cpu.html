Wat is de CPU?

De processor (CPU) is het brein van een computer die alle instructies uitvoert en ook berekeningen maakt om taken te doen. Het is een van de belangrijkste componenten van een computer. Bij elke nieuwe telefoon die uitkomt wordt er ook een nieuwere en betere CPU in de telefoon gedaan om de telefoon sneller en beter te maken.

Wat doet de CPU?

Een CPU voert dus instructies uit en maakt ook berekeningen, maar hoe werkt dit precies? Een cpu voert een driedelig proces uit.

• Fetch: Een CPU haalt een instructie op uit de RAM.

• Decode: De CPU ontcijfert de instructie.

• Exucute: De CPU voert de instructie uit.

CPU cores

Een CPU bevat één of meer CPU cores. Een CPU core kan aan één taak tegelijk werken. Dit vormt ook een fysieke beperking in de snelheid waarmee een CPU taken kan uitvoeren. Mijn laptop heeft een intel i5 CPU met 4 cores.
